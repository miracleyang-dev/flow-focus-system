# 心流 (Flow) - 个人效率系统

这是一个基于 Web 技术构建的个人效率系统，旨在帮助您保持专注并进入“心流”状态。该项目支持作为 PWA (Progressive Web App) 使用。

## 📁 项目结构

- `index.html` - 主应用界面
- `app.js` - 前端应用逻辑
- `style.css` - 样式文件
- `server.js` - 后端服务器逻辑 (Node.js)
- `data.json` - 数据存储/配置文件
- `manifest.json` - PWA 配置文件
- `启动心流.bat` - Windows 一键启动脚本

## 🚀 如何运行

1. 确保您的电脑上已安装了 [Node.js](https://nodejs.org/)。
2. 双击运行 `启动心流.bat` 即可一键启动服务并打开应用。
   或者，您也可以在终端中手动运行以下命令：
   ```bash
   node server.js
   ```
3. 打开浏览器访问控制台输出的本地地址（通常是 http://localhost:端口）。
