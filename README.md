# 🌊 心流 (Flow) - 个人效率系统

<p align="center">
  <b>沉浸式专注，高效管理你的每一分钟。</b>
</p>

这是一个极简且高效的个人专注力与效率管理应用。基于现代 Web 技术与 Node.js 构建，拥有 PWA (Progressive Web App) 支持，助你在喧嚣的工作和学习中快速进入"心流"状态。

## ✨ 核心特性

- 🎯 **沉浸专注**：极简界面设计，减少干扰，助您快速进入深度工作状态
- 📱 **PWA 支持**：支持安装到桌面或移动设备的主屏幕，离线可用，原生级体验
- ⚡ **轻量后端**：使用 Node.js 驱动，配合 Redis (ioredis) 实现高速数据读写与状态管理
- 💧 **水滴系统**：完成任务与习惯打卡获得水滴奖励，量化你的专注成果
- 🛒 **消费商城**：用水滴兑换自定义奖励，激励持续行动
- 🔄 **长期习惯**：支持反复完成的循环任务，记录时长/次数，持续积累
- 📊 **甘特图**：时间轴可视化任务分布，移动端专属垂直布局适配
- 🤖 **AI 陪伴**：智能任务分析、情绪支持、行动规划

## 🧩 功能模块

| 模块 | 说明 |
|------|------|
| **仪表盘** | 每日概览、统计卡片、今日任务、逾期提醒、常用链接 |
| **倒空大脑** | 快速输入待办事项，支持 AI 智能分析拆解或手动添加 |
| **任务看板** | 按优先级排列，支持拖拽排序（桌面&移动端）、标签筛选 |
| **甘特图** | 桌面端14天横向时间轴 / 移动端5天垂直卡片布局 |
| **AI 陪伴** | 基于任务上下文的对话助手，支持对话收藏 |
| **长期习惯** | 可反复完成的循环任务，记录时长或次数，自定义水滴奖励 |
| **消费商城** | 使用水滴兑换奖励商品，支持自定义商品 |
| **设置** | AI配置、标签管理、常用链接、数据导入导出 |

## 💧 水滴规则

完成任务获得水滴，公式：`floor(预估时间 / 60 × 优先级权重)`

| 优先级 | 权重 |
|--------|------|
| 重要紧急 | ×4 |
| 重要 | ×3 |
| 紧急 | ×2 |
| 一般 | ×1 |

长期习惯打卡同样获得水滴，每个习惯可自定义水滴数量。

## 🔄 长期习惯

- 独立模块，仅支持手动添加或导入（AI不会自动生成）
- 支持"时长"和"次数"两种记录方式
- 每次打卡可按自定义规则获得水滴
- 不做勾选结束，可反复完成

## ⏰ 每日刷新机制

- 默认刷新时间：**每日 00:00**
- 实现方式：
  1. `scheduleMidnightRefresh()` 精确计算距次日0点的毫秒数，setTimeout触发
  2. `visibilitychange` 事件兜底：设备休眠跨越0点后恢复时自动检查并触发
  3. 页面加载时检查 `lastDailyCheck` 是否为今日
- 刷新内容：循环任务日期推进并重置完成状态；清除已完成的过期非循环任务

## 🎨 界面特性

- **深色/浅色主题**切换
- **侧边导航可拖拽排序**（桌面拖拽 / 移动端长按拖拽）
- **移动端完整适配**：响应式布局、触摸手势支持
- **跨设备同步**：5秒轮询 + visibilitychange 即时同步

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, 原生 JavaScript（无框架依赖）
- **后端**: Node.js (基于 `http` 模块的原生服务)
- **存储**: Redis (主要数据源) + JSON文件备份，支持 localStorage 离线兜底
- **其他**: PWA (`manifest.json` & Service Worker `sw.js`)

## 🚀 快速开始

### 运行环境要求

- [Node.js](https://nodejs.org/) (建议 v14 及以上版本)
- [Redis](https://redis.io/) (可选，无 Redis 时使用 JSON 文件存储)

### 安装依赖

```bash
npm install
```

### 启动应用

```bash
npm start
# 或者: node server.js
```

启动后访问 `http://localhost:3000`

### ☁️ 云端部署 (如 Railway)

配置 `REDIS_URL` 和 `PORT` 环境变量后，可直接一键部署到 Railway 或其他 Node.js 托管平台。

## 📱 PWA 安装指南

1. 使用 Chrome 或 Edge 等现代浏览器打开本应用
2. 点击地址栏右侧"安装此应用"图标
3. 此后可像原生应用一样直接打开使用
